# Custom Packaging Boxes Website

Yeh ek modern aur responsive web application hai jo custom packaging boxes ke business ke liye banayi gayi hai. Ismein aapke customers products dekh sakte hain aur custom boxes ke liye quote (price) request kar sakte hain.

## Features

- **Responsive Design:** Mobile, Tablet, aur Desktop par behtareen dikhti hai.
- **Home Page:** Khubsurat hero section, key features, aur popular packaging categories.
- **Request a Quote Form:** Ek mukammal form jahan customers apni size aur quantity bata kar price pooch sakte hain.
- **Modern UI:** Material 3 design system ka istemal.

## Tech Stack

- Framework: Flutter
- Language: Dart
- Platforms: Web, iOS, Android, Windows, macOS, Linux

## Run Instructions

```bash
flutter pub get
flutter run -d chrome
```
